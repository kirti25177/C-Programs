# C Programs Collection

Welcome to my repository of C programs. This project is a collection of code snippets and scripts designed to cover fundamental programming concepts and logic-based problem solving.

## 🛠️ How to Run
To compile and run these programs, use a C compiler like `gcc`:

1. Open your terminal.
2. Compile the file: 
   `gcc max.c -o max`
3. Run the executable:
   `./max`

## 🚀 Goals
The goal of this repository is to build a solid foundation in C programming, covering topics like loops, arrays, pointers, and data structures.
